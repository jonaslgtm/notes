# 不到 300 元的高性价比真无线降噪耳机 - 倍思 SIMU S1 主动降噪耳机上手体验

在 AirPods 的引领下，TWS 蓝牙耳机深受消费者的喜爱，现如今不论是走到哪里都能发现带着无线蓝牙耳机的人，像我每天是坐地铁上下班的，地铁里大约 80% 以上的人都戴的是无线蓝牙耳机，办公室里至少一半的同事耳朵里塞着无线蓝牙耳机，你习惯在地铁/办公室听音乐吗？

近两年各大品牌厂商纷纷推出了自己的 TWS 蓝牙耳机，今天上手体验一款不到 300 元的高性价比真无线降噪耳机 - 倍思 SIMU S1 主动降噪 TWS 蓝牙耳机。

![倍思 SIMU S1](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-01.jpg)

## 一、外观设计

不得不说，倍思这款降噪耳机的颜值真的非常高，我入手的是经典的白色，它还有黑色、粉色、锖色、宝石蓝等配色可选，都非常的耐看。

![外观设计1](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-02.jpg)

充电仓光滑圆润，手感也很棒，整体大小与果厂 AirPods Pro 差不多。

![外观设计2](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-03.jpg)

电量指示灯设计在合盖处，不占地，但又很显眼，非常细节。

充电仓后面有一个圆形按钮，按一下就可以在这里看到电量情况，很方便。

## 二、佩戴体验

![佩戴体验](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-04.jpg)

这款倍思降噪耳机采用入耳式设计，单只耳机仅重 3.5g；

前后腔采用 180° 小仰角设计，导通耳朵内外气压平衡，长时间佩戴也很舒适；

包装内还附送了 2 款不同尺寸的硅胶耳塞，适应不同的耳道，视情况自行更换，我使用的是默认的耳塞，大小合适，狂甩不掉，日常生活、运动健身配戴都可以。

## 三、音质表现

![音质表现](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-05.jpg)

支持 AAC/SBC 解码，要知道，这个价位支持 AAC 解码的并不多。

最近几天使用倍思降噪耳机听了各种类型的歌曲，整体来说，音质表现还可以，三频比较均衡，10mm钕磁铁动圈结合高密复合振膜，重低音的表现不错。

把耳机给媳妇体验了，她听流行音乐，DJ 这类歌曲，昨天反馈给我说，比我手上 1000 元左右的蓝牙耳机听歌有感觉，我还能说什么。。。

## 四、降噪效果

重点来喽~

![降噪效果](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-06.jpg)

倍思降噪耳机采用 ANC 主动降噪技术，集合双核 DSP 高性能处理芯片，利用动圈单元发出的反向降噪声波，让噪音相互抵消从而实现降噪。

当第一次拿起耳机时就会自动开启深度消噪模式，可以有效削减环境噪音，不论是在地铁上，还是在办公室里，都能获得几分宁静。

它还有通透模式，长按(2秒)耳机柄感应区域即可完成切换，不用拿下耳机，就可以清晰听见周围的声音，与人交谈也不在话下，平时上班坐地铁，再也不怕坐过站了。

还有 ENC 通话降噪也能实现不错的通话质量。

**说下我个人主观体验：**

我平常通勤在地铁上的时候会开启深度消噪模式，实测可以过滤大部分噪音，出了站以后我会切换到通透或正常模式，毕竟路上人多车多，安全还是很重要的。总的来说，降噪效果还是不错的，可满足大多数人日常使用需求。

## 五、充电&续航

![充电&续航](https://cdn.jsdelivr.net/gh/jonaslgtm/zhimg@latest/p358757266-07.jpg)

耳机采用 Type-C 充电口，配合包装内的 Type-c 数据线使用，约 1.5 小时间即可充满；

耳机满电正常可用 4~5 小时，配合充电仓可用 20~24 小时；

以我正常的使用频率来说，2~3 天充一次电即可，这个续航能力还算可以。

## 六、其他

倍思降噪耳机搭载蓝牙 5.1 芯片 + LDS 激光镭雕天线，连接速度快、信号稳定，延时低，我用来吃鸡、王者、追剧基本上没感觉到有延时。

总结一下吧，倍思 SIMU S1 主动降噪耳机整体来说还是不错的，高颜值、高性价比，支持 ANC 主动降噪，音质也不错，续航能力也很强，重要的是价格不到 300 块，非常适合预算有限，想要体验主降噪的小伙伴。
